¿Que es el navegador web?

El programa informático llamado navegador web o comúnmente navegador, que permite a los usuarios acceder a las páginas web deseadas también se denomina navegador web siempre que conozcan la URL donde se encuentra (por ejemplo, www.google.com) o haga clic en el hipervínculo a esta página. Los navegadores web son programas muy utilizados hoy en día porque no se puede navegar por la web sin ellos. Algunos de ellos vienen preinstalados en nuestras computadoras
